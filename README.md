# 案例测试项目

## 项目说明

该项目用于测试各种第三方库。

## 项目启动

### 下载依赖
```
npm install / cnpm install / yarn
```

### 启动命令
```
npm run serve / dev / start
```

### 项目打包
```
npm run build
```
